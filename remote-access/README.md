## Nimani o'rganasiz

- **SSH tunneling**: local/remote/dynamic port forwarding, SOCKS proxy, jump host (bastion), reverse tunnel, ngrok/Cloudflare Tunnel — internetdagi mashinaga yoki NAT orqasidagi xizmatga xavfsiz kirish.
- **VPN**: qanday ishlaydi, WireGuard vs OpenVPN vs IPSec, split tunneling, o'z VPN'ingizni qurish, VPN vs proxy vs tunnel farqi.
- **Remote Desktop**: RDP (Windows), VNC, X2Go va NoMachine (Linux GUI) — grafik ish stoliga masofadan ulanish, farqlari, xavfsizligi.
