## Nimani o'rganasiz

- **Jo'natish**: email qanday ishlaydi (SMTP, MTA), qaysi service (Resend, SendGrid, Postmark, SES, Mailgun), SMTP vs API, transactional vs marketing.
- **Deliverability**: SPF, DKIM, DMARC, sender reputation, bounce, spam filtri — nega email spam'ga tushadi va qanday oldini olish.
- **Template**: HTML email nima uchun qiyin (Outlook!), MJML, React Email, responsive dizayn, testing.
- **Kampaniyalar**: list boshqaruvi, segmentatsiya, personalizatsiya, A/B test, unsubscribe/qonuniy talablar (CAN-SPAM/GDPR), metrikalar (open/click rate).
